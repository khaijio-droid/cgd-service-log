ออฟไลน์ (Local Storage)
💡 หลักการทำงาน: เมื่อตั้งค่า Google Sheets URL แล้ว
• ทุกครั้งที่ บันทึกข้อมูล → ส่งขึ้น Cloud ทันที
• ทุกครั้งที่ เปิดแอป → ดึงข้อมูลจาก Cloud มารวมอัตโนมัติ
• กดปุ่ม 🔄 Sync บนแถบด้านบนได้ตลอดเวลา
☁️ เชื่อมต่อ Google Sheets (ใช้งานร่วมกันออนไลน์)
✅ ขั้นตอนการตั้งค่าใช้งานร่วมกันออนไลน์
1. เปิด Google Sheets สร้าง Spreadsheet ใหม่
2. เมนู Extensions → Apps Script แล้ววางโค้ดด้านล่าง
3. กด Deploy → New Deployment → Web App
4. ตั้ง Execute as: Me, Who has access: Anyone → Deploy
5. คัดลอก Web App URL วางในช่องด้านล่าง แล้วกดบันทึก
